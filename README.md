# Create Wechat App

[![NPM Version][npm-image]][npm-url]

## Installation

```
$ npm i -gd create-wxapp
```

## Usage

```
$ create-wxapp init
$ cd <project-folder>
$ npm i -d
```

## Commands

- `npm run dev`
- `npm run build-debug`
- `npm run build-test`
- `npm run build-release`
